# Proyecto E-commerce Coder

Se implementa un aplicativo par e-commerce de productos de cosmética y utiliza las siguientes tecnologías:
- React
- Firebase

## Instalación

1. Versiones de soft
-   Node >= 22.11.0
-   Vite >= 6.2.0

2. Instalar dependencias

```
    npm install
```

3. Ejecutar proyecto

```
    npm run dev
```

## Librerías implementadas
- Firebase
- React-router-dom
- React-hook-form
